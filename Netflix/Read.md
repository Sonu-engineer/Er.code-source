# Netflix Landing Page Clone 🎬

This is a simple **Netflix landing page clone** built using **HTML and CSS**.  
It replicates the basic UI of Netflix's homepage with a background image, navigation bar, and email signup section.

---

## 📁 Project Structure


project-folder/
│── index.html
│── style.css
│── Netflix background.jpg
│── netflix log.png
│── README.md


---

## 🚀 Features

- Responsive landing page layout
- Background hero section
- Language selector dropdown
- Sign-in button
- Email input with "Get Started" button
- Styled using pure CSS (no frameworks)

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## 📄 How to Run

1. Download or clone this repository
2. Make sure all files are in the same folder:
   - `index.html`
   - `style.css`
   - Images (`Netflix background.jpg`, `netflix log.png`)
3. Open `index.html` in your browser
